# Project Overview - TICCI

## Executive Summary
TICCI is a B2B/B2C e-commerce platform built on **Laravel 9** using the **Botble CMS** framework. Initially launched as a generalist store, it is pivoting towards a **Value Added Reseller (VAR)** model, leveraging engineering expertise to offer optimized hardware, premium services, and B2B solutions.

## Technology Stack

| Component | Technology | Version | Notes |
|-----------|------------|---------|-------|
| **Framework** | Laravel | 9.21 | Core PHP Framework |
| **CMS** | Botble | 6.x (dev) | E-commerce Platform |
| **Language** | PHP | 8.0+ | Server-side logic |
| **Frontend** | Vue.js | 3.x | Reactive components |
| **Styling** | Sass / Bootstrap | 5.3 | Farmart Theme |
| **Database** | MySQL | - | Relational Data |
| **Build Tool** | Webpack Mix | 6.x | Asset compilation |

## Architecture
The project follows a **Modular Monolith** architecture standard to Botble:
- **Core**: Base system functionality (`platform/core`).
- **Plugins**: Modular features (`platform/plugins`).
- **Themes**: Frontend presentation (`platform/themes/farmart`).
- **Packages**: Shared libraries (`platform/packages`).

## Strategic Pivot
TICCI is transitioning from a standard e-commerce site to a specialized technical provider.
- **Focus**: "Tranquillité Numérique" (Digital Peace of Mind).
- **Key Features Planned**: Service Packs, Configurator, B2B Portal.
